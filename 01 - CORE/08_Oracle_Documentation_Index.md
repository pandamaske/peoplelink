## D. Excel Template Handoff (how to populate rows)

For each row in `GEODIS_Repositories_CORE HR_Template.v26.11.25.xlsx`:

- Repository: Use the exact repository name from the Methodology (Section 3). 
- Category: Organization Structure / Workforce Structure / Personal Details / Employment Details / Compensation / Process and Security.
- Key Oracle Docs: Choose from Section A mappings; add cross-cutting docs from Section B if needed.
- Dependencies: Refer to Methodology dependencies (e.g., LDG before Legal Entity, Legal Entity before BU, etc.).
- Validation Evidence: Reference expected outcomes (e.g., sample configuration screenshots, HDL load result, OTBI check, FSM task completion).
- Notes: Add country-specific exceptions or additional Oracle doc references when applicable.

Suggested columns (if needed): Repository | Category | Dependencies | Primary Docs | Supporting Docs | Evidence | Status
